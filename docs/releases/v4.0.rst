===========
Version 4.0
===========

.. note::

  v and up require Postgres version 13 or greater! (I.e. Ubuntu 20.04 will no longer work.)


**Version 4.0 adds support for editing 802.1Q tagged/trunk interfaces.**

This is initially support for the following drivers/devices:
- Arista devices via the eAPI
- Aruba AOS-CX via the API
- Juniper devices via the PyEZ API

See the documentation for more details.
