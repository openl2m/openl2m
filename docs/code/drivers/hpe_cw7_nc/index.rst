==================================
HPE CW7 NetConf Driver (Abandoned)
==================================

There is a skeleton implementation of an HPE Comware driver using the NetConf protocol,
located in *switches/connect/hpe_cw7_nc/connector.py*.

This driver is read-only, and only supports basic interface information. It is no longer supported,
and will be replaced by the HPE Comware REST API driver.

The driver code remains to service for educational purposes.

Old configuration info is still :doc:`here. <../../../configuration/hpe_cw7_netconf>`
